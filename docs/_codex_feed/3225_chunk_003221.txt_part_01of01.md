PCI Security Standards Council Bulletin: Technical FAQs for Secure Software Standard v1.x Published 09 May 2023 The PCI Security Standards Council (PCI SSC) has published the Technical Frequently Asked Questions (Technical FAQs) for Secure Software Standard v1.x. Technical FAQs are a mechanism for PCI SSC to provide additional clarification regarding the interpretation and application of security and program requirements between revisions of a given PCI security standard. Technical FAQs are an integral part of the Secure Software Standard and Program and must be fully considered during a Secure Software Assessment. This release of Technical FAQs is intended to address stakeholder inquiries regarding acceptable methods for satisfying control objectives within the Secure Software Standard, and to announce an extension to the deadline for Secure Software v1.1 submissions. This document is now available in the Document Library of the PCI SSC website. ###
