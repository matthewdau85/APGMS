Se prevé lanzar una nueva RFC entre septiembre y octubre de 2020. Lauren Holloway: Una vez que se publiquen PCI DSS versión 4.0, se otorgará a las organizaciones un periodo ampliado para la transición de la versión 3.2.1 a la versión 4.0. Nota: Se prevé finalizar la versión 4.0 seis meses antes de la publicación de las actualizaciones de la documentación justificativa, la capacitación y el programa requeridas para respaldar el uso de PCI DSS.
